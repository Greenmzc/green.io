开始使用hexo搭建博客

  • blog
  • shell

本博客是用Hexo搭建的，使用的是3.2.2版本。Hexo使用起来非常轻巧方便，只需敲入几个命令行就能生成简易博客，除此之外还能自己折腾插件和主题来丰富更多内容

下面介绍一下搭建过程，当然前提是要安装Git和Node

安装

全局安装Hexo

初始化

在本地进入博客目录，比如blog，然后初始化

此时目录下会生成必要的基础文件，包括配置文件(_config.yml)，初始化之后要安装npm包

生成静态页面

生成静态页面，会保存至public目录下，使用简写的(hexo g)也可以

本地启动服务

在浏览器输入localhost:4000打开就可以进行本地调试了，使用简写的(hexo s)也可以

部署博客

完成本地的操作后，下面要把博客推到github上了，首先需要在_config.yml配置文件配置github上git信息

注意冒号后面必须带个空格，否则无法被识别，接下来一个命令自动把本地的内容推送至github

现在可以在外网访问自己的博客了！

可能遇到的问题

  • 执行 hexo deploy 报 ERROR Deployer not found: git 错误

没有安装hexo-deployer-git，执行npm install hexo-deployer-git --save
